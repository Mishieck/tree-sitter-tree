# TreeSitterTree

A Zig library for converting tree-sitter trees to [Zig-native trees](https://github.com/mishieck/graphz).

## Features

<dl>
  <dt>Traversal</dt>
  <dd>Provides easy traversal of syntax trees.</dd>
  <dt>Mutability</dt>
  <dd>Tree-sitter trees are immutable. This library uses mutable trees.</dd>
</dl>
